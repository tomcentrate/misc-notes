#+TITLE: Spacemacs Frequently Used Shortcuts

* Terminology
  This is how to read the different shortcuts.
** M - alt
** C - ctrl
** Spc - Spacebar
** C-v = Ctrl+v
* Basic Emacs Usage
** Copy - ~M-w~
** Paste - ~C-y~
** Cut - ~C-w~
* Capture Stuff
** Copy Link of current file - ~C-c l~
** Insert Link - ~C-c C-l~
** Insert new Capture - ~Spc a o c~  
** Insert New Journal ~Spc a o j~
* Timekeeping
** Clock in - ~Spc m C i~
** Clock out - ~Spc m C o~
** Pomodoro - ~Spc m C p~
